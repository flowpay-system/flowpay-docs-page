# Standards

This template keeps standards intentionally light.

## What is enforced

- Markdown lint
- Prettier formatting for docs and config files
- Basic editor defaults from `.editorconfig`

## What is expected

- Prefer clarity over cleverness
- Keep repository structure easy to scan
- Add automation only when the project earns it
- Document decisions that affect future contributors

## How to extend

When a new project needs stricter rules, add them inside that project. Do not turn the base template
into a policy maze for problems that may never exist.
